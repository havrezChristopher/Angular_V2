À propos de notre projet

Notre projet est une application web moderne conçue pour répondre aux besoins de nos utilisateurs grâce à une combinaison de technologies avancées. Nous avons choisi des technologies de pointe pour le front-end et le back-end afin de garantir une expérience utilisateur optimale et des performances exceptionnelles.

Front-End

  • Angular: Un framework JavaScript robuste pour des interfaces utilisateur réactives et dynamiques.
  • Angular Material: Une bibliothèque de conception d'interface utilisateur basée sur Angular pour des interfaces esthétiques et conviviales.

Back-End

  • Microsoft SQL Server (MS SQL) : Un système de gestion de base de données relationnelle pour stocker efficacement nos données.
  • Sequelize Un ORM (Object-Relational Mapping) pour Node.js pour simplifier la manipulation des données de la base de données SQL.
  • Express.js Un framework Node.js pour créer des API REST et gérer la communication entre le front-end et la base de données.
